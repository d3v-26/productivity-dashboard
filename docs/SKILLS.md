# Skills (Claude Code Slash Commands)

Custom slash commands in `.claude/commands/`.

| Command | Purpose |
|---------|---------|
| `/scaffold` | Bootstrap the full Vite project from scratch |
| `/build-feature [habits\|todos\|calendar]` | Build one complete feature from docs |
| `/add-habit [name] [icon]` | Add a habit to the default seed data |
| `/review` | Audit all files against design system + data model |
| `/deploy` | Build, verify config, print deploy instructions |

See each `.claude/commands/<name>.md` file for the full prompt.
